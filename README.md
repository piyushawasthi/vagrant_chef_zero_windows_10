Vagrant + Windows 2010 + Chef_zero 
==================================

About
-----
This project shows creating a guest VM for windows 2010 with Vagrant and then provisioning it with Chef_zero and upload apache2 cookbook. 


How to use
----------

1. `vagrant up`
2. `vagrant provision`
